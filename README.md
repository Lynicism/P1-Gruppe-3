# P1-Gruppe-3
Github repo for P1 CCT AAU Gruppe 3 2025

Python file should probably also get an update with more user-friendly functionality.  

To start, you need to first start ArduinoGetSendData.ino, then after the Arduino starts functioning, you need to close ArduinoGetSendData.ino, and now you can open GetSaveData.c. After some use of the product, you can open test_get_file.py to visualize the data.

There is already MyFile.csv that contains random data for dates from 01-12-2025 to 17-12-2025, which can be used as it is, or can be deleted to make a new file
